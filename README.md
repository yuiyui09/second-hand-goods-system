# Second-Hand Goods System — ระบบจัดการฐานข้อมูลรับซื้อของเก่า

โปรเจกต์รายวิชา Database — แอปพลิเคชัน Windows Forms (C# / .NET) สำหรับจัดการฐานข้อมูลร้านรับซื้อของเก่า

## เทคโนโลยีที่ใช้
- **ภาษา:** C#
- **UI:** Windows Forms (.NET)
- **IDE:** Visual Studio

## โครงสร้างโปรเจกต์
```
UInew.sln            # Solution ของ Visual Studio
UInew/               # โปรเจกต์หลัก
  Form1.cs           # ฟอร์มหลักและ logic
  Form1.Designer.cs  # โค้ดออกแบบ UI (auto-generated)
  Program.cs         # จุดเริ่มต้นโปรแกรม
  Properties/        # Resources และการตั้งค่า
```

## วิธีเปิดโปรเจกต์
1. เปิดไฟล์ `UInew.sln` ด้วย Visual Studio
2. กด **Build** แล้ว **Run** (F5)

---
โปรเจกต์นี้จัดทำเพื่อการศึกษาในรายวิชา Database
